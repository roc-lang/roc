## Counter.roc
~~~roc
Counter := { count: U64 }.{
    new : {} -> Counter
    new = |{}| { count: 0 }

    increment : Counter -> Counter
    increment = |{ count }| { count: count + 1 }

    value : Counter -> U64
    value = |{ count }| count
}
~~~
